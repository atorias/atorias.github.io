# markdown

## 标题

```markdown
#       h1
##      h2
###     h3
####    h4
#####   h5
######  h6
```

## 段落

空白行分割上下文

```markdown
段落1
空白行
段落2
```

## 换行

末尾加两个空格

```markdown
你好呀[space][space]  
你好
```

## 强调语法

`**bold**`  
`**italic**`  
`***bold+italic***`

## 引用

```markdown
> 引用内容
```

嵌套引用

```markdown
> 第一层
>> 第二层
>>> 第三层
```

效果:
> 第一层
>> 第二层
>>> 第三层
